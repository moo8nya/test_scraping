# Scraping with BeautifulSoup4

## 1. Install Python3
1. Install homebrew
2. `brew update`, `brew upgrade` and `brew doctor`
2. `brew install python3`
3. `python3 -V`

## 2. Install Modules
* `pip3 install requests`
* `pip3 install lxml`
* `pip3 install beautifulsoup4`

## 3. Check Modules
* `pip3 freeze`

## 4. Run main.py
1. `python3 main.py`
